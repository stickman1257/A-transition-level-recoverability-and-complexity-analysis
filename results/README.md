# Result Summaries

This directory keeps lightweight outputs suitable for version control:

- `recoverability/`: architecture comparison summaries, integrated stage-pair table, and final figures.
- `complexity/`: permutation entropy/sample entropy transition summaries and sanity-check output.
- `predictability_horizon/`: auxiliary predictability-horizon summaries and figures.

Excluded artifacts:

- model checkpoints (`*.pt`, `*.pth`, `*.ckpt`)
- fold-level training logs
- full raw or interim EMG data
- local manuscript drafts
